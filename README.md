Aquesta pràctica consisteix a convertir una placa ESP32 en un servidor web capaç de monitoritzar una sala de servidors (CPD). El sistema llegeix dades de sensors (temperatura i llum), les mostra en una pàgina web i activa alertes visuals i un LED quan es detecten valors perillosos. També inclou una pàgina secundària amb crèdits i l’ús de GitHub per gestionar el desenvolupament.
